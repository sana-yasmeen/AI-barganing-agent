# AI Bargaining Agent – Smart Price Negotiator

## 📌 Project Overview

AI Bargaining Agent is a web-based e-commerce application that enables customers to negotiate product prices before purchasing. Instead of fixed pricing, users can submit counter offers, and the system automatically accepts or rejects them based on predefined bargaining rules.

The project was developed using **Python, Django, MySQL, HTML, CSS, and JavaScript**.


## ✨ Features

* User Registration & Login
* Product Search
* Product Filtering (Type, Brand, Style)
* AI-Based Bargaining
* Shopping Cart
* Secure Payment
* Order Tracking
* Feedback System
* Admin Panel for Product Management

## ⚙️ Installation

1. Clone the repository.
2. Install the required packages:

   ```
   pip install -r requirements.txt
   ```
3. Open MySQL and execute the `AIBargainingAgent.sql` file.
4. Update the database credentials in `settings.py` if required.
5. Run the project:

   ```
   python manage.py runserver
   ```
6. Open the browser and visit:

   ```
   http://127.0.0.1:8000/
   ```


## 📁 Database

The SQL file included in this repository creates the **AIBargainingAgent** database and the required tables:

* register
* addproduct
* customer_order
